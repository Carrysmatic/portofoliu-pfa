# 🚀 Digital Portfolio - SaaS Solutions on Subscription

This project represents the central platform for presenting digitalization services for local businesses (PFA). The focus is shifted from "code" to **business results, automation, and efficiency**.

---

## 🛠️ Technology Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) (Modern configuration via `@import`)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** TypeScript
- **Interactivity:** React Hooks (`useState`, `useEffect`) for real-time simulations.

---

## ✨ Implemented Features

### 1. Home Page (Hero & Concept)
- **High-Impact Design:** Asymmetric layout with refined typography (Next.js 15 + Tailwind v4).
- **Phone Simulator:** An interactive component in the Hero section demonstrating the "Digital Guestbook" interface (WiFi, Check-out, Local Guide).
- **Selling the Benefit:** Clear messages addressing the problem of repetitive processes.

### 2. Strategy "How I Can Help" (`/solutions`)
- **Solutions Catalog:** Defined the 6 SaaS directions (Guestbook, Maintenance, NPS Feedback, vCard, WMS, Ticketing).
- **Non-Technical Language:** Content adapted for entrepreneurs (focus on time saved, not technologies).

### 3. Business Model (Subscription)
- **Recurring Revenue:** The entire site promotes long-term partnerships through monthly subscriptions.
- **Transparency (FAQ):** Dedicated section to address fears about hidden costs and technical support.

### 4. Solutions Lab (Split-Screen Demo)
- **Dashboard Manager (Laptop):** Simulated control panel receiving notifications.
- **Client Simulator (Mobile):** Interactive interface where users can "request the WiFi password" or "report a defect."
- **Real-time Sync:** Animations and loaders simulating latency and the operation of a real system.

### 5. Layout & Design
- **Navbar:** Glass effect (`backdrop-blur`) and fixed positioning.
- **Responsive:** Adapted for Desktop, Tablet, and Mobile.
- **Tailwind v4 Setup:** Clean structure of the `globals.css` file.

## 📦 Installation

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/username/portofoliu-pfa.git
   cd portofoliu-pfa
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Run the Project in Development Mode:**
   ```bash
   npm run dev
   ```
   Access the app at `http://localhost:3000`.

4. **Build for Production:**
   ```bash
   npm run build
   npm start
   ```
